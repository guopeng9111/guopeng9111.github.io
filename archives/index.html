docker
2023年06月14日 230 字 大概 1 分钟

1、在安装docker之前，先初始化机器环境，如果之前安装过旧版本的docker，应该先使用命令进行卸载。
yum remove docker
docker-client
docker-client-latest
docker-common
docker-latest
docker-latest-logrotate
docker-logrotate
docker-selinux
docker-engine-selinux
docker-engine
docker-ce

2.Yum源配置
备份配置文件
mv /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo.backup
wget -O /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-7.repo
wget -O /etc/yum.repos.d/epel.repo http://mirrors.aliyun.com/repo/epel-7.repo
yum clean all
yum makecache

3.docker安装
（1）需要虚拟机连网，安装yum工具
yum install -y yum-utils
device-mapper-persistent-data
lvm2 –skip-broken

（2）更新本地镜像源
um-config-manager
–add-repo
https://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo
sed -i ‘s/download.docker.com/mirrors.aliyun.com/docker-ce/g’ /etc/yum.repos.d/docker-ce.repo
yum makecache fast

（3）docker安装
输入命令：yum install -y docker-ce

Docker安装完毕！

2023年06月13日 319 字 大概 1 分钟

一、（创建虚拟环境）
1.安装Python环境到windows上
Windows系统配置Anaconda
打开资料中提供的:Anaconda3-2021.05-Windows-x86_64.exe文件
第一步：上传Spark安装包
1.对于安装包的获取，共有两种途径方法。
1)方法一、通过md文档中提供的开源地址进行自主下载。
下载地址
https://dlcdn.apache.org/spark/spark-3.2.0/spark-3.2.0-bin-hadoop3.2.tgz
2)方法二、通过老师所给的链接，从网盘下载相应组件安装包。

2.上传这个Spark安装包文件到Linux服务器中

解压下载的Spark安装包

tar -zxvf spark-3.2.0-bin-hadoop3.2.tgz -C /export/server/
配置环境变量：
#HADOOP_HOME
export SPARK_HOME=/export/server/spark
#PYSPARK_PYTHON
export PYSPARK_PYTHON=/export/server/anaconda3/envs/pyspark/bin/python
#HADOOP_CONF_DIR
export HADOOP_CONF_DIR=$HADOOP_HOME/etc/hadoop
#ZOOKEEPER_HOME
export ZOOKEEPER_HOME=/export/server/zookeeper
export PATH=$PATH:$ZOOKEEPER_HOME/bin
export JAVA_HOME=/export/server/jdk1.8.0_2
重置环境变量
Source /etc/profile
PYSPARK_PYTHON和 JAVA_HOME 需要同样配置在:/root/.bashrc中

上传文件到Linux服务器中
解压文件
tar -zxvf spark-3.2.0-bin-hadoop3.2.tgz -C /export/server/

设置软链接:
ln -s /export/server/spark-3.2.0-bin-hadoop3.2 /export/server/spark

进去pyspark

测试

打开服务器4040：
